# viaMulher
viaMulher é um app desenvolvido em React Native, a ideia do app é disseminar a informação sobre violencia domestica no Brasil, e com isso, diminuir os casos de feminicidio. o App informa canais de denuncias, exibe o ciclo de violencia contra a mulher e abre a possibilidade do usuario responder ao Formulario de Avaliação de Risco(FRIDA), assim, com as respostas, o formulario pode servir de material de apoio aos tecnicos que forem fazer escuta especializada.
